indexOf()를 이용하여 검색하기

HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym : acronym 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • address : address 태그는 웹 페이지의 연락처 정보를 설정합니다.
  • aaplet : aaplet 태그는 는 자바 애플릿을 보이게 하는 곳에 쓰입니다.
  • area : area 는 이미지의 핫스팟 영역을 정의하고 하이퍼링크를 추가할 수 있습니다.
  • article : article 태그는 사이트의 독립적인 컨텐츠 섹션을 설정합니다.
  • aside : aside 태그는 주요 콘텐츠의 보조적 컨텐츠 섹션을 설정합니다.
  • audio : audio 태그는 오디오 파일을 설정합니다.
  • b : b 태그는 다른 텍스트와 구별 할 때 설정합니다.
  • base : base 태그는 모든 문서의 기본이 되는 URL을 설정합니다.
  • basefont : basefont 태그는 문서의 기본 폰트, 사이즈, 종류를 설정합니다.
  • bdi : bdi 태그는 텍스트의 출력 방향 영역을 설정합니다.
  • bdo : bdo 태그는 텍스트의 방향을 설정합니다.
  • big : big 태그는 텍스트의 크기를 크게 설정합니다.
  • blockquote : blockquote 태그는 긴 문장의 인용문을 설정합니다.
