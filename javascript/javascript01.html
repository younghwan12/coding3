01. 변수 : 데이터 저장

변수는 데이터를 저장하는 저장소 입니다. 이 저장소에는 숫자, 문자, 함수, 객체 등을 저장할수있습니다. 

var x = 100;
var y = 200;
var z = "javascript";

document.write(x);
document.write(y);
document.write(z);
