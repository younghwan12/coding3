01. 선언적 함수

선언적 함수는 함수를 선언할 때 이름을 붙여주는 함수를 말한다. 즉, 따로 변수에 넣어주지 않고 선언할 대 붙여준 이름으로 호출하면 된다. 

function func(){
    document.write("함수가 실행되었습니다.1
");
}
func();
결과보기
함수가 실행되었습니다.1

02. 익명 함수

익명 함수는 함수명 대신 변수명에 함수 코드를 저장하는 구현 방식이다. 익명 함수의 소스 코드는 변수값이므로 끝에 세미콜론 ; 을 대입한다. 익명 함수는 호출 시 변수명을 함수명처럼 사용하면 된다. 

// 변수안에 함수가 들어있어서 이름이 없다.
const func = function(){
    document.write("함수가 실행되었습니다.2");
}
func();
결과보기
함수가 실행되었습니다.2

03. 매개변수 함수

함수의 매개변수(parameter)란 함수를 호출할 때 인수로 전달된 값을 함수 내부에서 사용할 수 있게 해주는 변수입니다. 

// str은 parameter, 매개변수를 이용한 함수
// 매개변수 함수에서 변수선언은 생략가능
// 변수 : 지역변수 + 전역변수 + 매개변수
// 함수 : 지역함수 + 전역함수
function func(str){
    document.write(str);
}
func("함수가 실행되었습니다.3");
결과보기
함수가 실행되었습니다.3

04. 리턴값 함수

리턴값이란 함수가 어떠한 기능을 수행하고 그 결과를 호출한 곳으로 돌려주는 값을 말합니다. return을 사용합니다. 

function func(){
    const str = "함수가 실행되었습니다.4
";
    return str;
}
document.write(func());
결과보기
함수가 실행되었습니다.4

05. 화살표 함수(선언적 함수)

선언적 함수를 화살표 함수로 바꾼작업입니다. 주석표시를 아래식처럼 생략해서 작성할수도 있습니다. 

// func = () => {
//     document.write("함수가 실행되었습니다.5");
// }

func = () => document.write("함수가 실행되었습니다.5
");            

func();

08. 선언적 함수

리턴값 함수를 화살표 함수로 바꾼작업입니다. 주석표시를 아래식처럼 생략해서 작성할수도 있습니다. 

// function func(){
//     const str = ("함수가 실행되었습니다.8

// }
// document.write(func());

func = () => {
    const str = ("함수가 실행되었습니다.8
")
    return str;
}

document.write(func());

09. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값

화살표함수(익명함수로) 틀을짠뒤 리턴값(결과)을 넣고, 매개변수 출력 

const func = (str) => {
    return str;
}
document.write(func("함수가 실행되었습니다.9
"));
결과보기
함수가 실행되었습니다.9

10. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값 + 괄호 생략

화살표함수(익명함수로) 틀을짠뒤 리턴값(결과)을 넣고, 매개변수 출력한다. 변수가 하나이므로 괄호 생략 

// 09번과 구조는 똑같지만 변수가 하나일때는 괄호가 생략이 가능하다.
const func = str => {
    return str;
}
document.write(func("함수가 실행되었습니다.10
"));
결과보기
함수가 실행되었습니다.10

11. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값 + 괄호 생략 + 리턴생략

10번과 같은구조이고 리턴도 생략이 가능하다. 

// 10번과 구조는 똑같지만 리턴을 생략해서 한줄이되어 위로 올라간다.
const func = str => str;

document.write(func("함수가 실행되었습니다.11
"));
결과보기
함수가 실행되었습니다.11

12. 화살표 함수 : 선언적 함수 + 매개변수 + 리턴값 + 괄호 생략 + 리턴생략

11번과 같은구조이고 익명함수를 선언적함수로 바꿀수도 있다. 

// 11번과 구조는 똑같고, 익명함수를 선언적함수로 바꿔서 const 생략 가능
func = str => str;

document.write(func("함수가 실행되었습니다.12
"));
